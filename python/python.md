#### 多版本管理
pyenv

> **如果网络不太好，用pyenv下载会比较慢。可以自己到官网下载，然后将文件放在`～／.pyenv/cache`目录下（不存在，新建即可）。pyenv会检查文件的完整性，若确认无误，则不会再重新下载，直接解包编译安装。**
>
